>_ Patchwork
paste code + error → get a fix

Paste a broken snippet and the error or traceback it produces. Patchwork sends both to Claude, returns a corrected version of your code, and explains what changed and why.

How it works

  1. Pick the language your code is written in.
  2. Paste the broken code into "Your code."
  3. Paste the error message or traceback it produced (or leave it blank if you just want a bug check).
  4. Tap "Fix my code" — Claude reads both, finds the issue, and sends back a corrected version with a plain-language explanation of what changed.

Fixes are suggestions, not guarantees — always review the corrected code before using it. Each browser gets 10 fixes / day, resetting at midnight.
