Drop in a snippet and the error or traceback it produces. Patchwork sends both to Claude, returns a corrected version of your code, and explains what changed and why.

How it works

01

Pick the language your code is written in.

02

Paste the broken code into the left pane.

03

Paste the error or traceback on the right (optional).

04

Run it — get back a fix plus a plain-language explanation.

Fixes are suggestions, not guarantees — always review corrected code before using it. Each browser gets 10 fixes / day, resetting at midnight.
