# Patchwork

Fill in a new client's details — name, project, price, timeline — and Patchwork uses Claude AI
to draft a professional contract and a matching invoice, ready to copy or save as a PDF.

🔗 Live: https://patchwork-rho.vercel.app

## How it works
1. Fill in your client's name, project, and price
2. Add timeline and payment terms if you have them
3. Generate — Claude drafts a contract and invoice
4. Copy the text or save each as a PDF, then send it to your client

Free to use — 10 drafts per day per browser.

Drafts are a starting point, not legal advice — have a professional review anything high-stakes
before you send it.
