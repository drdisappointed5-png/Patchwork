# Patchwork

Paste broken code + the error it produces, and Patchwork uses Claude AI 
to return a fixed version with a plain-language explanation of what changed.

🔗 Live: https://patchwork-rho.vercel.app

## How it works
1. Pick your language
2. Paste your broken code
3. Paste the error/traceback (optional)
4. Get a fixed version + explanation

Free to use — 10 fixes per day per browser.
